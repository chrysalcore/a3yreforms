## Description

**A3Y Reforms — Landing Page** provides a single-page website for A3Y Reforms, a company offering construction, maintenance, repair, electrical, plumbing, and related services in and around Roanoke, Virginia.

The site is organized into sections including **Services**, **Completed Projects**, **Work Process**, and **Contact**, using a vibrant color palette that reflects the company's brand.

The project is implemented with React, component-based structure, and BEM-style CSS. It follows best practices for responsiveness and accessibility.

## Main Features

- Advanced state management with React hooks.
- Modular components written in JavaScript.
- Responsive UI: styles adapt across device sizes.
- Smooth scrolling and a fluid user experience.

## Local Installation

Follow these steps to run the project on your local machine.

### Prerequisites

- [Node.js](https://nodejs.org/es/)
- [NPM](https://www.npmjs.com)

### Steps

1. Clone the repository

    ```bash
    git clone https://github.com/chrysalcore/a3yreforms.git
    cd a3yreforms
    ```

2. Install dependencies

    ```bash
    npm install
    ```

3. Run development mode

    ```bash
    npm run dev
    ```
