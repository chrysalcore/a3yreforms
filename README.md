## Description

**A3Y Reforms Landing Page** was created to provide a website for the organization of the same name. This organization offers construction, maintenance, repair, electrical, plumbing, and other services throughout the Roanoke city, in Virginia.

The site features a single page with several sections, including **Services**, **Completed Projects**, **Work Process**, and **Contact**. All sections utilize a vibrant color palette characteristic of the company.

The code was written using React, state management, componentization, and the BEM methodology, adhering to best practices, responsiveness, accessibility, and other web standards.

## Main Features

- Advanced state management.
- Components with JavaScript.
- Responsive UI/UX: Interface designed with CSS that adapts to all devices.
- Smooth scrolling and a fluid user experience.

## Local Installation and Use

Follow these steps to run the project on your local machine.

### Prerequisites

- [Node.js](https://nodejs.org/es/)
- [NPM](https://www.npmjs.com)

### Steps

1. Clone the repository

    ```bash
    git clone https://github.com/chrysalcore/a3yreforms.git
    cd a3yreforms
    ```

2. Install dependencies

    ```bash
    npm install
    ```

3. Run development mode

    ```bash
    npm run dev
    ```
